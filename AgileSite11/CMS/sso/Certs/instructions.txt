Generating X509 cert:

if you have git for windows install there is a version of Open ssl here:
C:\Program Files\Git\usr\bin\openssl.exe

convert .pfx to pem file (private key only)
eg. openssl pkcs12 -in .\trainingnetwork.pfx -out .\trainingnetwork.pem -nocerts

Generating X509
convert .pfx to pem with only public key
eg. openssl pkcs12 -in .\trainingnetwork.pfx -out .\trainingnetwork.pub.pem -nokeys
convert pem to crt (X509)
eg. openssl x509 -outform der -in .\trainingnetwork.pub.pem -out .\trainingnetwork.pub.crt

openssl pkcs12 -in .\2022-trainingnetworknow.pfx -out .\2022-trainingnetworknow.private.pem -nocerts
openssl pkcs12 -in .\2022-trainingnetworknow.pfx -out .\2022-trainingnetworknow.pub.pem -nokeys
